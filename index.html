Run the full Java app locally.

GitHub Pages hosts this static showcase. The live Servlet app needs a Java runtime and database.

  1. 1. Clone the repository Use SSH after adding your GitHub key.
  2. 2. Start Docker Compose MySQL is initialized with the schema and demo data.
  3. 3. Open the local app Visit the Tomcat route exposed by Docker. 
git clone git@github.com:Mayank-Chaubey/ProductivityTracker.git
cd ProductivityTracker
docker compose up --build

# App URL:
# http://localhost:8081/ProductivityTracker
GitHub Pages is static hosting. JSP pages, servlets, authentication, JDBC, and MySQL run only in a Java hosting environment.
